SharpHook

SharpHook provides a cross-platform global keyboard and mouse hook, event simulation, and text entry for .NET. It is a wrapper of libuiohook and provides direct access to its features as well as higher-level types to work with it.

The following articles can help you learn how to use this library.

GitHub Repository

https://github.com/TolikPylypchuk/SharpHook

Installation

dotnet add package SharpHook --version 5.0.0
dotnet add package SharpHook.Reactive --version 5.0.0

Supported Platforms

SharpHook targets .NET 6+, .NET Framework 4.6.1+, and .NET Standard 2.0. The following table describes the availability of SharpHook on various platforms:

Windows macOS Linux
x86 Yes N/A No1
x64 Yes Yes Yes
Arm32 No2 N/A Yes
Arm64 Yes Yes Yes

[1] - Linux on x86 is not supported by .NET itself.

[2] - Windows Arm32 support was dropped in .NET 5.

libuiohook only supports X11 on Linux. Wayland support may be coming, but it's not yet here.
