Interface IReactiveGlobalHook

Represents a global reactive keyboard and mouse hook.

RunAsync()

Runs the global hook without blocking the current thread. The hook can be destroyed by calling the Dispose() method.

Declaration
IObservable<Unit> RunAsync()
Returns
Type Description
IObservable<System.Reactive.Unit>

An observable which is completed when the hook is destroyed.
Remarks

Since the underlying native API for running a global hook is blocking, the only way to run it without blocking the current thread is to run it on a separate thread.

The returned observable is hot, and emits a single value and then immediately completes when the hook is destroyed.

Exceptions
Type Condition
HookException

Starting the global hook has failed.
InvalidOperationException

The global hook is already running.
ObjectDisposedException

The global hook has been disposed.
