Delta

The wheel delta.

Declaration
public ushort Delta
Field Value
Type Description
UInt16

The wheel delta.
Remarks

On Windows this value is equal to the WHEEL_DELTA constant (120).

On macOS this value is the number of pixels per line.

On Linux this value is always equal to 100.
