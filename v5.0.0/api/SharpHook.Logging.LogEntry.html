Remarks

For simple handling the Level and FullText properties are anough. For more advanced handling you can use the entries' formats and arguments.

Each log entry from libuiohook has the following format: 

function [line]: message

The function and line reference the source code of libuiohook itself, and can be retrieved using the Function and SourceLine properties. The message can be retrieved by splitting the text by : and getting the trimmed second part.

You shouldn't create LogEntry instances yourself. Instead, use the LogEntryParser class to create them in the logging callback.
