Class EmptyLogSource

Represents a souce of libuiohook logs which never raises any events.

public sealed class EmptyLogSource : ILogSource, IDisposable
Remarks

This class exists primarily to be used instead of LogSource in release builds of client apps as they will most probably not need any logs from libuiohook. Also, unlike LogSource this class doesn't need the Visual C++ Redistributable package to be installed on Windows.
