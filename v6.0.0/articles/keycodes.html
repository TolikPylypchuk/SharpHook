Key Code Mappings

The following table contains the mapping of virtual key codes defined in the SharpHook.Native.KeyCode enum to OS-specific key code definitions. Note that many key codes are not available on all 3 OSes.

Important

Key code values in the KeyCode enum are meaningless and may change between major versions. Only the enum constant names should be taken into consideration (e.g. when persisting key codes).

Sources:

  • Windows: Microsoft Docs.
  • macOS: HIToolbox/Events.h as defined in macOS 13.5 with some additional key codes included in libuiohook.
  • X11: /usr/share/X11/xkb/keycodes/evdev as defined in Kubuntu 22.04
  • Evdev: input-event-codes.h from the Linux source code.
Note

Key codes marked with O are obsolete since these keys don't exist on modern keyboards. These key codes will most probably be removed in the next major version.
